Pricing Cards Website

This project is a modern, high-end Pricing Cards website built with HTML, CSS, and JavaScript.
It features responsive design, glassmorphism style, interactive pricing cards, animated price counters, dark/light mode, and a testimonial section.

Features
1. Pricing Cards

Three plans: Basic, Pro, Enterprise.

Popular plan highlighted with a badge.

Hover animations: scale and shadow effect.

Tooltips for disabled features.

Animated price counters.


2. Dark/Light Mode

Toggle button to switch between light and dark mode.

Animations for smooth color changes.

Fully responsive for mobile and tablet devices.

3. Testimonial Section

Customer testimonial with avatar, name, and position.

Glassmorphism style matching the pricing cards.

Hover animation for smooth interaction.

4. Responsive Design

Fully responsive on all screen sizes: desktop, tablet, and mobile.

Cards automatically adjust to single column on small screens.

Fonts, icons, and buttons scale appropriately.

Testimonial adapts to screen width.

5. Interactive Features

Clicking on a card highlights the selected plan.

Tooltip on disabled features appears on hover.

Price counter animation for visual appeal.

Scroll fade-in effect for cards.
